# workout_API
Aqui estão somente os comandos essenciais para criar o projeto, sem os touch:

⸻

1. Criar pasta do projeto e iniciar o Node

mkdir api-entregas-veiculos
cd api-entregas-veiculos
npm init -y


⸻

2. Instalar dependências

Produção:

npm install express

Desenvolvimento:

npm install -D typescript ts-node-dev @types/node @types/express


⸻

3. Inicializar o TypeScript

npx tsc --init


⸻

4. Criar estrutura de pastas

mkdir -p src/routes src/controllers src/services src/data


⸻

5. Configurar scripts (você edita no package.json)

(sem comando, porque é edição manual)

⸻

6. Rodar o servidor

npm run dev


⸻

Se quiser, te mando a implementação base do index.ts, rotas e services.
